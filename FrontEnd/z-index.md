# code

```css
/* 字符值 */
z-index: auto;

/* 整数值 */
z-index: 0;
z-index: 3;
z-index: 289;
z-index: -1;/* 使用负值降低优先级 */

z-index: inherit;
z-index: initial;
z-index: unset;
```
