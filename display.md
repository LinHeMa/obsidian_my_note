# 定義
用來定義`flex container` ，有了這個標簽，就可以讓這個標籤以內的子元素都成為`flex items`  

## note
	Note that [[[[CSS]]]] columns have no effect on a flex container.

```[[css]]
.container { display: flex; /* or inline-flex */ }
```
